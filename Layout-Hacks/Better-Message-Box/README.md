## Installation Instructions
> [!IMPORTANT]
> NSMBWer+ 1.2 and later already use the improved retail layout, so there is no need to install it.

Decide which of the two layouts you want to use, and rename it to `msgbox.arc`. Then, move it into your
layout folder. Newer calls this folder `Layouts`, while NSMBWer+ calls it `Layout`.

After that, its installed and nothing else needs to be done!

> [!TIP]
> If using the retail-styled layout in Newer, its recommended to also delete the `MessageFont` out of the `Font` folder.
> The layout was designed to be used with the original font, so using Newer's may cause minor issues.
